このアプリについて

iCloud カレンダーを、自分のサーバー上でシンプルに閲覧・検索できるプライベートな Web アプリです。 Apple の公式 Web インターフェースでは難しい細かいカスタマイズや、 深夜帯の折りたたみ・フルテキスト検索といった自分だけに必要な機能を自由に追加できることを目標に開発しました。

CalDAV プロトコルを通じて iCloud からカレンダーデータを取得し、 FullCalendar v6 を用いてブラウザ上に描画しています。 認証情報はサーバー上の環境変数ファイルにのみ保持し、外部へは一切送信しません。

開発の経緯

iCloud.com のカレンダー画面は機能的ではあるものの、動作が重くなりがちで、 深夜の予定が多い日に表示範囲を手動でスクロールするのが煩わしいという不満が開発のきっかけです。

「CalDAV で取得できるなら Flask で表示するだけ」という単純な出発点から始まり、 Claude Code（Anthropic の AI コーディングアシスタント）との対話を重ねながら 機能を段階的に拡充してきました。 設計・実装・デバッグのほぼ全工程を Claude Code との協働で進めており、 本ページの内容も同様です。

開発の変遷

  • 2026年5月11日
    初回リリース
    Flask + FullCalendar v6 の組み合わせで基本機能を実装。
    • 月 / 週 / 日 / 一覧 ビュー切り替え
    • 複数カレンダーの色分け表示・凡例
    • イベントクリックで詳細モーダル（場所・メモ）
    • 深夜帯（21:00〜5:00）の折りたたみ表示
    • ダークモード対応
    • スマートフォン向けレスポンシブレイアウト
    • Nginx + Gunicorn によるサブパス（/calendar/）配信
  • 2026年5月20日
    検索機能・サジェスト機能の追加
    予定のフルテキスト検索と、入力補助のためのサジェストドロップダウンを実装。
    • タイトル・場所・説明・カレンダー名を対象としたキーワード検索
    • 入力欄フォーカス時に頻出ワードをドロップダウン表示
    • 入力に合わせてリアルタイムフィルタリング（↑↓キー選択対応）
    • ワード抽出は全角記号・波ダッシュ等の日本語デリミタに対応
    • 初回検索は直近2ヶ月に絞り高速化、ボタンで過去1年〜2年先に拡大可能
